## Use

- Install Django python library
  ```bash
  pip3 install django
  ```

- Create a superuser account for the site management directily from the web page
  ``` bash
  python manage.py createsuperuser
  ```
- Run the server with
  ```bash
  python manage.py runserver
  ```
